GroundForge - Diagrams of bobbin lace patterns

Make variations on predefined patterns (tutorial)

  • Select a pattern from a catalogue page, they link back to this page with predefined patterns.
  • Choose stitches (examples) with the bold symbols in the prototype diagram.
  • Apply the wand (faster) or link button (saves the pattern in the browser history), details.
  • Choose contrasting threads with the square starts or bobbins in the thread diagram, more colors.
  • Start the animation of the pair diagram to make the holes as round as in the thread diagram.
