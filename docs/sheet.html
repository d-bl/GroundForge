GroundForge

Pricking prototypes for bobbin lace grounds.

Patterns on the thumbnail page have two links: one to change stitches on thread/pair diagrams the other to choose pin positions on corresponding pricking prototypes. The latter is on this page. Pick one if you happen like it, or download and edit one into a pricking in the desired style. Most important clue about reshaping patterns from the user guide: dots with the same color should match when "sewing" repeats together. These dots should also move together, a process made easy by state of the art general purpose drawing editors.
