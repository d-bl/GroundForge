Cheat sheet

Some less trivial stitches to copy-paste that may produce interesting results when using the thread diagram as pair diagram for the level two diagrams.

A second string is the mirrored version of the stitch.

= crclct, clcrct

= clcrclc, crclcrc

= ctctc

= ctclctc, ctcrctc

= ctclcrctc, ctcrclctc

= ctcttctc
